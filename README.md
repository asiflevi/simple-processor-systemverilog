# Simple Processor – SystemVerilog

## Overview
A small processor implemented in SystemVerilog, demonstrating modular RTL design
and verification using a dedicated testbench.

## Modules
- `proc.sv` – Top-level processor module
- `regn.sv` – Register module for data storage
- `dec3to8.sv` – 3-to-8 decoder for control logic
- `tb_proc.sv` – Testbench for functional verification

## Design Highlights
- Modular RTL architecture
- Decoder-based control logic
- Testbench-driven verification

## Technologies
- SystemVerilog
- RTL design
